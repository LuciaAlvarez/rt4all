# Ubuntu

### Instalar electron
```shell
curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install npm
```

### Ejecutar
```shell
~$ electron .
```
**o**
```
~$ npm start
```
